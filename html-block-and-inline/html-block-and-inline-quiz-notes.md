# html-block-and-inline-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- How do block-level elements affect the document flow?

They always start on a new line, making the element appear vertically on the page instead of horizontally.

- How do inline elements affect the document flow?

They appear horizontally, flowing from left to right instead of from top to bottom.

- What are the default width and height of a block-level element?

Stretches 100% wide (all of the width of it's parent element) for it's width and for it's height is equal to the height of its contents.

- What are the default width and height of an inline element?

Equal to the height and width of it's content.
